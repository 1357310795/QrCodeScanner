## 简易二维码扫描器

C# WPF 原生开发，在电脑上识别二维码。
![](screenshot\QQ截图20210912113002.png)

## 功能丰富
 - 支持四种模式：截图识别+摄像头识别+本地图片识别+作为扫描枪使用
 - 支持Zbar和Zxing两种主流引擎
 - 支持多码同扫
 - 支持Material Design缤纷主题色与暗黑模式
 - 支持保存上一次的设置
 - 独创的扫描枪模式
